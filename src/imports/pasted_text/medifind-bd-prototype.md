Design a modern, professional, and realistic desktop web application prototype for "MediFind BD", a Medicine Availability, Price Comparison, Reservation, and Pharmacy Management System for Bangladesh.

The system connects patients with registered pharmacies. Each pharmacy has its own dashboard to update medicine inventory, prices, and availability. Medicine availability shown to users is based on the inventory updated by registered pharmacies.

The UI should follow modern healthcare design principles similar to Arogga, Practo, 1mg, and Google Maps. Use Bootstrap 5 style with rounded cards, clean typography, soft shadows, and a blue, green, and white healthcare color palette.
